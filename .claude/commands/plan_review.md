# Review Implementation Plan

**First, ensure we're up to date:**
```bash
git fetch
git status
```

Confirm and display the current feature branch name.

---

**Then review the plan:**

Please review the implementation plan for the current feature or issue.

**Focus on:**
- Completeness of implementation steps
- Appropriate level of detail
- Opportunities for simplification (KISS principle)
- Test coverage strategy
- Potential risks or blockers
- MCP protocol compliance
- Type safety considerations

Please provide feedback on any complexity that could be reduced and any questions or suggestions you have.
