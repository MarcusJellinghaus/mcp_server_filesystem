Add _apply_pr_merge_override() pure function + tests

Add a pure function that overrides local rebase status when GitHub
confirms the PR is mergeable (squash-merge safe). This prevents
misleading "rebase needed" recommendations when the PR can be
merged directly.

- Add _apply_pr_merge_override() to branch_status.py
- Add TestApplyPrMergeOverride with 4 test cases covering:
  - Passthrough when rebase not needed
  - Override when rebase needed but PR is mergeable
  - No override when PR is not mergeable (False)
  - No override when PR mergeability is unknown (None)
