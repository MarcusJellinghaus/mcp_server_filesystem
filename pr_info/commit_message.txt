feat: add verify_github() with connectivity checks (#157)

Add CheckResult TypedDict and verify_github() orchestrator function
that runs checks 1-4 independently:
- Check 1: GitHub token configured (with OAuth scopes reporting)
- Check 2: Authentication via get_user()
- Check 3: Repository URL resolvable from git remote
- Check 4: Repository accessible via GitHub API

Each check runs in its own try/except for independent reporting.
overall_ok reflects only error-severity checks.
