feat: add output filtering module for read-only git operations

Add structure-aware filtering and truncation for git command output:

- filter_diff_output(): parses unified diffs and returns only hunks
  matching a search pattern, with configurable context lines
- filter_log_output(): splits git log into commit entries and returns
  only entries matching a search pattern
- truncate_output(): truncates text to a line budget with a notice

All functions are pure (string in, string out) and case-insensitive.
Comprehensive unit tests with synthetic diff/log data included.

Part of #77 (read-only git operations).
