feat: add search_files content and combined search modes

Add content search (regex) and combined (glob + regex) modes to
search_files. Includes context lines support, max_result_lines
truncation cap, binary file skipping, and regex validation.

- Content-only mode: search all files with regex pattern
- Combined mode: glob filters files, then regex searches content
- context_lines parameter includes surrounding lines in matches
- Binary files silently skipped on UnicodeDecodeError
- Invalid regex raises ValueError upfront via re.compile()
- Dual cap: max_results and max_result_lines (whichever hits first)
- 7 new tests covering all content search functionality
